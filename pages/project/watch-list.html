Watch-list

Sauvegardez vos films pour les regarder plus tard!

Voir le site

Dans le cadre d’un projet personnel j’ai codé une application qui permet à l’utilisateur de créer une liste de films, qu’il aimerait regarder, ou qui lui aurait été conseillés. J’utilise l’API The movie Db pour afficher toutes les informations sur les films. Quand l’utilisateur ajoute un film à sa Watchlist, je le crée dans la base de donnée en utilisant les informations fournies par l’API sur le film.
